# Инструкции по билду проекта

### 1) Убедитесь, что у вас заполнены frontend/.env и backend/.env

### 2) Перейдите в папку /deploy

--------------------------------

## Сборка в dev режиме

- ### 3) Перейдите в папку dev

- ### 4) Выполните команду ```docker compose up -d```

--------------------------------

## Запуск в production режиме

- ### 3) Перейдите в папку prod

- ### 4) Выполните команду ```docker compose up -d```
